Audacity 3.2.1

        Duble Click: Select Full Track
        Home: Goto start
        P: Pause/Play
        R: Record
        Spacebar: Play/Stop
        Ctrl+1: ZoomIn
        Ctrl+3: ZoomOut
        Ctrl+X: Cut
        Ctrl+C: Copy
        Ctrl+V: Paste
        Ctrl+I: Split Clip
        Ctrl+L: Silence audio selection
        Ctrl+Z: Undo
        Ctrl+Shift+Z: Redo
        Ctrl+O: Import Audio
        Ctrl+Shift+I: Import Audio in new path
        Shift+U: Mute/Unmute
        Shift+R: Record New Track
        Select, Delete: Delete
        Fit selection to width: Ctrl+E
        Fit Project to width: Ctrl+F

        
Remove Noise

        1: Select some noisy part
        2: Goto: /Effect/Noise Removal and Repair/Noise Reduction/Get Noise Profile
        3: Select All, /Effect/Noise Removal and Repair/Noise Reduction/OK

        
Filter Curve EQ

        1: Select All
        2: Goto: /Effect/EQ and Filters/Filter Curve EQ/Presets & settings/Factory Presets/Bass Boost/Apply
        3: Goto: /Effect/EQ and Filters/Filter Curve EQ/Presets & settings/Factory Presets/Treble Boost/Apply

        
Compression

        1: Select All
        2: Goto: /Effect/Volume and Compression/Compressor/:-
            Threshold: -12 dB;
            Noise Floor: -40 dB;
            Ratio: 2:1;
            Attack Time: 0.20 secs;
            Release Time: 1.0 secs;
            Make-up gain for 0 dB after compressing
        
        
Normalize

        1: Select All
        2: Goto: /Effect/Volume and Compression/Normalize/:-
            Remove DC offset [center on 0.0 vertically];
            Normalize peak amplitude to: -1.0 dB;
