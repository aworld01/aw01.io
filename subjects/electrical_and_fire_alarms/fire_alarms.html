Fire Alarm System

What is Fire_Alarm_System (FAS)?:-

Fire panel's name:-
(1) Eaton
(2) Notifire
(3) Adwards

How many type of Fire_Alarm_Systems?:-
There are two type of Fire_Alarm_System:
(i) Addressable Fire_Alarm_System
(ii) Conventional/None-Addressable Fire_Alarm_System

What are the devices in Fire_Alarm_System?:-
(01) Fire Alarm Panel / FACP (Fire Alarm Control Panel)
(02) Repeater Panel
(03) Smoke Detector
(04) Heat Dectector
(05) Flame Detector
(06) Multi sensor
(07) Menual Call Point (MCP)
(08) Manual Pull Station (MPS)
(09) Hooter
(10) Sounder
(11) Isulator Module (IM)
(12) Monitor Module (MM)
(13) Relay Module (RM)/ Control Module (CM)

Classes of wiring in Fire_Alarm_System?:-
(i) Class A in Addressable Fire_Alarm_System.
(ii) Class B in Conventional Fire_Alarm_System.

Class A = Loop wiring
Class B = Zone wiring

Cable used in Fire_Alarm_System:-
2c*1.5 smm Armoured FRLS (Fire Retardant Low Smoke) cable.

Type of Smoke Detector?:-
(i) Ionization (ii) Photoelectric (iii) Comination of Ionization and Photoelectric

Distance between devices:-
SD: 7.5m Radius
HD: 5.3m Radius
SD to SD: 10m HD to HD: 07m Corridor 15m maximum

Note: MCP/MPS should be always install in corridor and stair case area.
