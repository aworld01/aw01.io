Audacity 3.2.1

        Ctrl+Shift+I: Import Audio
        Duble Click: Select Full Track
        Mute/Unmute: Shift+U
        P: Pause/Play
        Spacebar: Play/Stop
        Ctrl+1: ZoomIn
        Ctrl+3: ZoomOut
        R: Record
        Shift+R: Record New Track
        Ctrl+X: Cut
        Ctrl+C: Copy
        Ctrl+V: Paste
        Ctrl+Z: Undo
        Ctrl+Shift+Z: Redo
        Ctrl+I: Split Clip
        Fit selection to width: Ctrl+E
        Fit Project to width: Ctrl+F

        
Remove Noise

        1: Select some noisy part
        2: Goto: /Effect/Noise Removal and Repair/Noise Reduction/Get Noise Profile
        2: Select All, Ctrl+R (/Effect/Repeat Noise Reduction)
        
        
Normalize

        /Effect/Volume and Compression/Normalize/:-
            Remove DC offset [center on 0.0 vertically];
            Normalize peak amplitude to: -1.0 dB;

        
Compression

        1: Select All
        2: Goto: /Effect/Volume and Compression/Compressor/:-
            Threshold: -7 dB;
            Noise Floor: -40 dB;
            Ratio: 3.5:1;
            Attack Time: 0.10 secs;
            Release Time: 1.0 secs;
            Make-up gain for 0 dB after compressing

        
Reverb

        1: Select All
        2: Goto: /Effect/Delay and Reverb/Reverb/:-
            Room Size: 60 %;
            Pre-delay: 10 ms;
            Reverberance: 50 %;
            Damping: 40 %;
            Tone Low: 100 %;
            Tone High: 100 %;
            Wet Gain: -2 dB;
            Dry Gain: -1 dB;
            Stereo Width: 100 %;compressing

        
Graphic EQ

        1: Select All
        2: Goto: /Effect/EQ and Filters/Graphic EQ/:-
            leftSide = Base
            rightSide = Treblecompressing

        
Base and Treble

        1: Select All
        2: Goto: /Effect/EQ and Filters/Base and Treble/:-
            Base: 3.0 dB;
            Treble: 5.0 dB;
            Volume: 2.0 dB;

        
Eco

        1: Select All
        2: Goto: /Effect/Delay and Reverb/Eco/:-
            Delay time: 1 secs;
            Decay factor: 0.06;
